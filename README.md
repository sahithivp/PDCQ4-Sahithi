# PDCQ4-Sahithi

Application for google authentication, and session management using Python Flask, and basic HTML, CSS for the UI styling. 
This application has 2 Phases in it.

## Phase 1: Authentication
- Google OAuth Authentication
- Displaying the profile picture, name and email id which has been signed in successfully
- Displaying current indian time
